

# Game

A C++ game project developed by Anyya__127, hosted on Gitee.

## 项目简介

这是一个使用 C++ 开发的项目，已配置好 Visual Studio Code 开发环境。

## 开发环境

- **编辑器**: Visual Studio Code
- **语言**: C/C++
- **编译器**: GCC/Clang/MSVC（根据系统选择）

## 使用方法

1. 确保已安装 C++ 编译环境
2. 使用 VS Code 打开本项目
3. 安装必要的 C/C++ 扩展插件
4. 编译运行代码

## 目录结构
